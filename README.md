# Projeto de Gerência de Configuração: UltraWiki
UltraWiki é uma ideia de projeto de fãs para o jogo Ultrakill — desenvolvido por Hakita e publicado pela NewBlood — que cobre alguns elementos importantes da obra, como inimigos, arsenal, e como os bônus de estilo que impactam o ranking do jogador em cada nível.

O projeto foi feito por estudantes do IFPB — Campus Cajazeiras (Danilo de Oliveira e José Henrique Abílio Lacerda).
